# Frontend
- **Framework**: React 19
- **Language**: TypeScript 5.9
- **Build Tool**: Vite 7
- **Styling**: Bootstrap 5 & React-Bootstrap
- **Visualization**: Chart.js & React-Chartjs-2
- **State/Communication**: React Hooks, SSE (Server-Sent Events) for streaming.

# Backend
- **Framework**: FastAPI 0.118+
- **Language**: Python 3.12+
- **ORM**: SQLAlchemy
- **Migrations**: Alembic
- **AI Orchestration**: LangChain (Google, Groq, Cerebras integrations)
- **Data Validation**: Pydantic 2.12+

# Database
- **Primary Database**: PostgreSQL

# Infrastructure & Tooling
- **Containerization**: Docker & Docker Compose
- **Python Package Manager**: uv
- **Node Package Manager**: pnpm
- **Security**: JWT Authentication, Fernet encryption for API keys.
